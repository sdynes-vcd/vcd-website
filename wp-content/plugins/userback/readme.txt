== Description ==

Get feedback from your WordPress website with screenshots from Userback. Download the plugin to enable Userback on specific, all or draft pages only.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> Userback screen to configure the plugin

== Frequently Asked Questions ==

= Is Userback free to use? =

Yes, there is a Free Plan for you to try out most of the features in Userback. Free Plan doesn't expire.

= What does Userback do? =

Userback lets you collect feedback from your web projects with real time screenshots and instant notifications.

= Does Userback work in local development environments? =

Yes, absolutely.

= Can I customize the Userback widget? =

Yes, Userback widget can be fully customized directly in the app or via JavaScript API.

= Can Userback be integrated with other platforms? =

Yes, Userback integrates with a full range of third party platforms: JIRA, HipChat, Slack, WordPress, Email...etc.

= Can I programmatically send additional data to Userback? =

Yes, you can use Userback's JavaScript API to easily send your custom data along with each feedback.

= Do I have to embed the widget code to be able to use Userback? =

No, you can install Userback's browser extension which allows you to collect feedback from any websites.
